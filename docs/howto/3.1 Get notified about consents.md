# Get notified about changes to consents

By calling the Consent Feed API, you can retrieve relevant changes to your consents, for example recently revoked or expired consents.

## Prerequisites

You know [how to authorize](consent_authorize.md) and retrieve access tokens. You have collected consent from an individual.

## 1. Retrieve access token

Use `audience=https://api-preprod.norskgjeld.no/feed/v1/consent` and `scope=client.access.consent.events`.

## 2. Poll consent events feed

Read an event stream from `/feed/v1/consent` with parameters `fromSequenceNr` and `limit`.

```json
{
  "startSequenceNr": 1,
  "endSequenceNr": 2,
  "generatedAt": "2024-05-21T12:51:57.383110992Z",
  "result": [
    {
      "sequenceNr": 1,
      "eventType": "SAMTYKKE_OPPRETTET",
      "consentId": "c8ff2ad5-4140-481d-a7cd-6c7deee7203c"
    },
    {
      "sequenceNr": 2,
      "eventType": "SAMTYKKE_ARKIVERT",
      "consentId": "c8ff2ad5-4140-481d-a7cd-6c7deee7203c"
    }
  ]
}
```
