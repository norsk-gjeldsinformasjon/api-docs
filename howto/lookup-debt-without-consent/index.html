    Look up debt without consent¶

    This service is for financial institutions with a legal basis under Gjeldsinformasjonsloven to request debt information without explicit consumer consent.

    Legal basis required

    Your legal/compliance function must confirm a valid legal basis before using this service. Without it, use consent-based lookups.

    Prerequisites¶

    • Signed agreement with Norsk Gjeldsinformasjon — contact post@norskgjeld.no
    • Valid enterprise certificate (SEID 2.0) and mTLS configured — see Security requirements
    • Network access to Norsk Gjeldsinformasjon test and production endpoints

    Endpoint¶

    Base URLs

    • Test: https://ws.preprod.norskgjeld.no
    • Production: https://ws.norskgjeld.no

    The search endpoint is at /api/rest/v2/debt/search and the OpenAPI spec is at /api/rest/swagger.json.

    Request¶

    POST with Content-Type: application/json; charset=UTF-8

    Parameter Required Description
    ssn YES Valid Norwegian NIN or D-number (11 digits)
    referenceID YES Your traceability reference, e.g. a loan application ID
    messageToCustomer No Message to notify the applicant. No identifying info. Max 100 chars.
    coBorrowers No Array of co-borrowers. Each item has one field: ssn.
    maxTimeToWait No Max wait in ms (1000–60000). Default: 3000.
    onBehalfOfOrganization Conditional Required if performing lookup on behalf of another FI.
    usingOwnNotification No Set true to disable Norsk Gjeldsinformasjon default notification. Requires separate agreement.

    Example request¶

    {
  "ssn": "01017012345",
  "referenceID": "APP-2024-0001",
  "messageToCustomer": "Application for unsecured loan",
  "coBorrowers": [],
  "maxTimeToWait": 3000,
  "usingOwnNotification": false
}

    Status codes¶

    Code Meaning Action
    200 Success Parse response, log, continue
    204 No data found Treat as "no debt" per your credit policy
    400 Input validation error Do not retry. Fix input and resubmit.
    403 Not authorised Check certificate and config. Contact Norsk Gjeldsinformasjon if unresolved.
    5xx Server error Retry with exponential backoff.
